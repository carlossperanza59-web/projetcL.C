README - Sistema de Login e Controle de Estoque

Este projeto implementa um sistema simples de Login, Cadastro de Usuários e Controle de Estoque utilizando Streamlit e SQLite3. 
Ele é ideal para pequenas empresas que desejam gerenciar usuários e produtos de forma prática e eficiente.

* Estrutura do Projeto
- login.py → Arquivo principal do sistema
- usuarios.db → Banco de dados para armazenar os usuários
- estoque.db → Banco de dados para armazenar os produtos
- usuarios.txt → Registro simples com os usuários cadastrados


* Funcionalidades

- Login de usuários com autenticação
- Cadastro de novos usuários
- Armazenamento de credenciais em banco SQLite e backup em arquivo TXT
- Cadastro de produtos no estoque
- Visualização de produtos em tabela estilizada
- Módulo de vendas com histórico

* Como Executar
1. Instale as dependências necessárias:
   pip install streamlit

2. Execute o sistema com:
   streamlit run login.py

* Observações
- Este projeto foi desenvolvido para fins de aprendizado.
